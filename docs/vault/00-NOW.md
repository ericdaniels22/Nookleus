- **Supabase MCP `get_logs` (service: postgres) is the fastest path to root-cause a redacted 500.** Surfaced during 2026-05-06 15d Test 1. The route returned `{ error: "internal error" }` (the `apiDbError` redactor in `src/lib/api-errors.ts` strips Postgres messages from client responses on purpose to avoid leaking schema details / RLS policy text).
